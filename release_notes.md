# Release Notes

## v0.0.3 - ?
* Add support for extracting [ASSB](https://www.ibm.com/docs/en/zos/latest?topic=iar-assb-information) and [ASXB](https://www.ibm.com/docs/en/zos/latest?topic=iar-asxb-information) control block data.
* Add support for filtering repeated control block data using [filters](./filters).
* Create separate Library/Executable so that CBXP can be invoked by [C/C++](./interfaces/c_cpp) callers.

Initial alpha release.
