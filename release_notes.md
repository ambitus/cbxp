# Release Notes

## v0.0.3 - ?
* Add support for extracting [ASSB](https://www.ibm.com/docs/en/zos/latest?topic=iar-assb-information) and [OUCB](https://www.ibm.com/docs/en/zos/latest?topic=rqe-oucb-information) control block data.
* Add support for filtering repeated control block data using [filters](../filters).
* Create static library distribution so that CBXP can be invoked directly by [C/C++](../interfaces/c_cpp) callers.
